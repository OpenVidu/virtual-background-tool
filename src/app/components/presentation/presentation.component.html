Welcome to the website to create a dataset for creating computer vision algorithms allowing you to replace your background in any video conference.

Virtual Background Feature

Virtual background is a very useful feature to change the real background with a sintetic background without having a green chroma background. It improves privacity if you connect from your own house. Also allows to provide a branded image for corporate usage.



Machine Learning

We will request you to record several videos in typical situations when you are in a videoconference. These videos will be used to train machine learning algorithms so they can be applied in real time when other people want to change the background when participating in a video conference.
